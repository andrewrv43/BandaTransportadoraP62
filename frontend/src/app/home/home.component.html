Uso de Sensores

Sensor TCS3200

El sensor TCS3200 es un sensor de color que puede detectar una amplia gama de colores. Funciona emitiendo luz a través de una lente y luego recogiendo la luz reflejada por el objeto. El sensor tiene una matriz de fotodiodos que convierten la luz en corriente eléctrica.

ESP32

El ESP32 es un microcontrolador de bajo costo y bajo consumo de energía que se utiliza comúnmente en proyectos de IoT (Internet de las cosas). Es altamente versátil y tiene capacidades de conectividad Wi-Fi y Bluetooth, lo que lo convierte en una opción popular para proyectos inalámbricos.

Arduino

Arduino es una plataforma de código abierto que consta de hardware y software. Es muy utilizada en el ámbito de la electrónica y la robótica para crear proyectos interactivos. El lenguaje de programación utilizado en Arduino es una variante de C/C++.

Sensor Ultrasónico

Un sensor ultrasónico utiliza ondas de sonido de alta frecuencia para medir distancias. Emite un pulso de sonido y mide el tiempo que tarda en recibir el eco reflejado desde un objeto. Con esta información, se puede calcular la distancia entre el sensor y el objeto.



SERVICIO WEB

Angular

Angular es un framework de desarrollo de aplicaciones web de código abierto y mantenido por Google. Permite crear aplicaciones web dinámicas y de una sola página (SPA) con una estructura modular. Angular utiliza TypeScript como lenguaje de programación y ofrece una amplia gama de características, como enlace de datos bidireccional, inyección de dependencias y manejo de rutas.

Express

Express es un framework de desarrollo web para Node.js que se utiliza para construir aplicaciones web y APIs de forma rápida y sencilla. Es minimalista y flexible, lo que permite a los desarrolladores crear servidores con facilidad. Express proporciona una variedad de herramientas y middleware para gestionar solicitudes HTTP, manejar rutas y realizar otras tareas comunes en el desarrollo web.
